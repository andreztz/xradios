# xradios

> Listen to your favorite internet radio stations with xradios.


[![See the demo video]()](https://github.com/andreztz/xradios/issues/2#issue-988180396)

## Installation

Linux and Windows:

```bash
$ pip install xradios
```

## Usage example

```bash
$ xradios
```

## Development setup

```bash
$ git clone git@github.com:andreztz/xradios.git
$ pip install -e .
```
